## 說明
做了一個網頁音效板，在波普貓愛吃魚2023生日時送給觀眾
- 網頁參考別人的css，加上一點vue做點擊標題到一定數量就逐個觸發的彩蛋
- 用GitHub Pages 提供靜態網頁
- 剪輯好的音檔放在Google Cloud Storage上
- 2023/07/21 因為GCS試用期間到期，故暫時關閉了
- 2023/12/04 實驗性的播放清單
- (未完成) RWD & Dark Mode
- 預覽播放
- 讀網址參數
